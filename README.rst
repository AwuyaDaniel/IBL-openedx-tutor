IBL-openedx-tutor plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/AwuyaDaniel/IBL-openedx-tutor.git

Usage
-----

::

    tutor plugins enable IBL-openedx-tutor
